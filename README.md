# flames

flames is a simple fun package to run the classic game Flames

## Installation

```bash
git clone https://github.com/AadilVarsh/flames.git
```

## Usage

```python

import flames

flame = flames.Flame('John Doe', 'Jane Doe')

print(flame.find_it())
```
