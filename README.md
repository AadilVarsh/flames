# pyflames

Pyflames is a simple fun package to run the classic game Flames

## Installation

```bash
pip3 install Pyflames
```

## Usage

```python

import pyflames

flame = pyflames.Flame('John Doe', 'Jane Doe')

print(flame.find_it())
```
